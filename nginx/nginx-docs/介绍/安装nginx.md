# 安裝 nginx

nginx 可以以不同的方式安裝，安裝方式取決於當前的操作系統。

## 在 Linux 上安裝

對於 Linux，可以使用 nginx.org 的 nginx [套裝軟體](../其他/linux包.md)。

## 在 FreeBSD 上安裝

在 FreeBSD 上，可以從 [套裝軟體](http://www.freebsd.org/doc/handbook/pkgng-intro.html) 或通過 [ports](http://www.freebsd.org/doc/handbook/ports-using.html) 系統安裝 nginx。ports 系統提供更大的靈活性，提供了各種選項可供選擇。port 將使用指定的選項編譯 nginx 並進行安裝。

## 從原始碼安裝

如果需要某些特殊功能套裝軟體和 ports 無法提供，那麼可以從原始碼中編譯 nginx。雖然此方式更加靈活，但對於初學者來說可能會很複雜。更多資訊請參閱 [從原始碼構建 nginx](../How-To/從原始碼構建nginx.md)。
